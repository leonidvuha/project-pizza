# Project Pizza

Project Pizza is a fullstack web application for a small pizza business.

## Idea

The application will allow customers to:

- view the pizza menu
- add pizzas to the cart
- register and log in
- place an order
- track the order status

The application will also include a simple admin panel for business owners.

## Planned features

### Public pages

- Home page
- Menu page
- Login page
- Registration page

### Customer features

- Cart
- Checkout
- Order status page

### Admin features

- Orders management
- Products management
- Change order status

## Tech stack

- Next.js
- TypeScript
- Tailwind CSS
- PostgreSQL
- Prisma
- shadcn/ui

## Project status

This project is currently in the initial setup stage.

## Getting started

Install dependencies:

```bash
npm install
```
